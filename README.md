🏨 Hotelus

Hotelus es una plataforma moderna para la gestión y reserva de hoteles, diseñada para ofrecer una experiencia rápida, intuitiva y escalable tanto para usuarios como para administradores.

🚀 Features
🔍 Búsqueda de hoteles en tiempo real
🏨 Visualización de habitaciones disponibles
📅 Sistema de reservas
👤 Autenticación de usuarios
💳 Gestión de pagos (opcional / integrable)
📊 Panel de administración
🌐 Interfaz responsive (mobile + desktop)
⚡ Alto rendimiento y UX optimizada
